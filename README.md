# 🚀 Absensi - Sistem Manajemen Kehadiran Karyawan

## 📝 Deskripsi
Sistem manajemen kehadiran karyawan ini terintegrasi dengan [backend Laravel](https://github.com/okedane/Absensi-Backend-Laravel).
